# VSS

VSS stands for **Very Simple Syntax**.

This workspace contains the first implementation skeleton for the VSS programming language in C.

## Current contents

- language specification
- compiler architecture notes
- starter C project structure
- lexer implementation skeleton

## Build

```bash
make
```

## Run lexer demo

```bash
./vss examples/hello.vss
```

This first version prints the token stream for a source file.
